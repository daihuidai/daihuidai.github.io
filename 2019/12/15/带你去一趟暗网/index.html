带你去一趟暗网

发表于 2019-12-15 | 分类于 科学上网 | | 阅读量 次

万壑松风知客来，摇扇抚琴待留声

1. 文起

说明：为了让小白也能了解暗网，本文将不深入说明任何相关技术的实现，可能会提供外网链接。

最近和一位女生聊到了一个话题——《暗网》，似乎很多人对这个灰色产业还是很感兴趣的。还记得我在大一刚搭建完科学上网服务后所做的第一件事情就是到暗网去逛了逛，对其本身并不是很感兴趣。能进暗网这是技术的一种体现，所以在进入过一次后就再也没有接触了。

最近抽了点时间打算对这个”神秘”的网站做一个简单的介绍。其实一般技术能访问到的并不是暗网，而是更上一层较为容易进入的交易网站——《深网》（也就是上图中的红色箭头）。

不论是深网还是暗网都属于非法产业，各国政府和相关机构从来都是在进行打压中。如果你是一个经常使用网络的人，那在你应该可以了解到暗网就是一个搭建：贩卖毒品、器官交易、色情交易、地下赌博、暗杀组织、机密盗取等等各种违法交易并且使用虚拟货币，如比特币来进行交易的平台。

具体是这样的吗？下面我带你进入真实的暗网看看它的初貌。

2. 什么阻挡了你进入暗网

首先如果要进入暗网，我们得了解暗网是一个怎样的平台，它是如何使用的。

2.1. 长城防火墙

想过一个问题没有，既然绝大多数人都知道暗网，那为什么绝大多数人都没有进入过，而且政府机关也没有禁止的办法？

（格外补充）作为一个中国人，从你开始使用网络的时刻起，我们伟大的祖国就免费强制的送给了你一个礼物——长城防火墙（GFW）。它既有好处也有坏处，坏处是它几乎杜绝了我们大陆与外界的联系，例如你无法使用最好的搜索引擎 Google，你无法使用最大的社交平台 Fackbook，无法第一时间查阅优质的国外资料等等。好处是国家政治是一个十分敏感的问题，如果你能够查看到国外的网站，你就会知道那些煽动暴乱、搞独立、抹黑中国领导人的文章铺天盖地。中国人口大国，教育素质和个体差异导致有极大一部分人员是无法区别这些误导信息的，说白了就是保护大多数国人（当然也有为了封锁消息）。它的存在将我们的世界限制和保护在了神奇的大陆——中国。

并且在国家的监管下，例如百度等搜索引擎会过滤掉敏感词汇。说了这么多，这些都是非暗网技术带来的你所不能深入了解暗网的原因。那暗网本身又是如何限制我们进入它们的领域的呢？下面给你展示暗网的访问入口是什么样。

2.2. 暗网本身的技术

上面这张图中红色的网址其实就是暗网的访问连接。你可能有点疑惑，这些链接怎么与我们平时见到的连接并不一样，它的中间都是乱码，结尾则是 onion 。其实这是暗网链接进行加密后的结果，普通人拿到了这些链接并没有任何用处，因为所有正常合法的浏览器都不能解码这些链接，会导致访问失败。需要使用专门的解析浏览器才能使用。

2.3. 洋葱网络

以上为第一个防范措施。下面说说另一个更重要的技术，这个技术不仅暗网在使用，例如美国军方、其它政府机构都会使用该技术——洋葱网络

你知道洋葱是什么样的吗？暗网就像我们生活中吃的洋葱，（洋葱炒肉很不错的）美丽的洋葱被外壳一层一层的包裹，拨开一层又一层，就像在隐藏着自己的内心不让别人触碰。

暗网则是利用了该技术，我们最初访问或传输的信息被这个网络想一层一层的蜕掉外壳，每次蜕掉一层它就给你下一层的位置，当你到达下一层，它又重复该过程，直到这个信息被传到无人能追查的地方。

所以现在知道为什么它能如此嚣张，任何国家都无法完全禁止它了吧。

3. 进入暗网

补充了这么多关于暗网的小知识，下面可以进去瞧瞧了。

3.1. 使用 Tor 浏览器

想要进入暗网，需要一个强大的工具，帮助我们解决上面的一些问题，它就是一个隐私、开放的自由网络工具——Tor Browser。

3.3. 暗网总结

看了暗网这个灰色产业的一小部分，想必你现在已经有了一个简单的认知，无非就是一群坏人为了挣钱通过某种技术开展的违法交易活动。

暗网这个空间大概占比我们所有网络的 4% ，是一个很小的一部分，这也是大多数人并不能接触到的原因。如果一个人内心没有邪恶的思想，那我们普通人是一辈子都不会和它有任何的交集。所以这就是一个可有可无的小插曲，出于好奇，我们了解了它。出于人性和自我保护，我们应当远离它。

4. 文末

暗网并不是一个多神奇的东西，反而是一个制造混乱的地狱。我不想过多的记录太多十分消极的内容，还记得刚进入暗网时看了一篇文章，一个人是如何被活活打死的，这让我十分的抵触，血淋淋的图片和文字让人想吐。

所以我们更应该学会如何保护自己，人类就应该探究世界，了解世界的未知，才能有应对的办法。
