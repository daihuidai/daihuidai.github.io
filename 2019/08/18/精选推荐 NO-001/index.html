1. proxyee-down

Proxyee Down 是一款开源的免费软件，基于本软件的高速下载内核和扩展，可以方便并快速的下载所需资源。Proxyee Down 属于 http 下载工具，基于 http 代理，支持多连接分块下载。

在作者的 GitHub 中你能找到源码仓库，README 文件简单介绍了该软件的基本情况。通过使用教程或 Wiki 你能找到如何快速使用该软件的方法和一些 BUG 解决方法。比如 Windows、Linux、Mac 用户如何下载和使用方法。

作者的官网已经介绍的足够详细，目前最新版本为 3.4，并且停止了更新。不过作者开了 GO 语言版本的仓库，目的是打造一个体积小、跨平台、内存低、可扩展、免费的下载器，本人表示很期待~~。

Proxyee Down 给我影响最深的是前面离线安装 Python 第三方库时，极大的提高了下载速度保证了任务的完成。

2. 倒带

倒带是一款免费的音乐播放器，开发者将网易云、QQ音乐的后台数据结合，可以登录网易云账号，无缝对两者的后台歌曲数据进行查询在线播放、clone（委婉下载）。

界面干净，收费、不收费的歌曲都可以播放、下载哦。

选择歌曲后同样可以查看滚动歌词、翻阅网易云或 QQ 音乐的评论，很多的小功能都是很实用的。

可以看得出做这款软件的作者是一个开发技术一流并且爱听音乐的程序员~~。如果你也喜欢音乐赶紧上网下载吧。
